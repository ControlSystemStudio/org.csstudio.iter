Archive XY Graph

An extension of XY Graph that enable to plot historic data from datasource

Special Properties

Behavior properties

  • Time Span
    Time span in seconds for historical data.
  • Archive Data Source
    URL of the data source.
  • Plot Data Source
    If yes, the URL of the data sources are the ones defined for the data browser ordered by KEY. If no, a new entry field is shown in the Property view to specify the URL of the data source "Archive Data Source"
