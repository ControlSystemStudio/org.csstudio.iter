Logbook
This tool allows entering messages into the logbook:
  1. Enter your user name and password for accessing the logbook
  2. Select a logbook (in case more than one is offered)
  3. Enter title of the new entry
  4. Enter the main text of the entry
  5. Optionally, add images (see below)
  6. Press 'Submit'
Images can be added via the "Add Image" button. Depending on the operating system you can also drag-and-drop images from the desktop, the OS X 'finder', or the Windows File Explorer into the gray image area.
